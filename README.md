# Brief Descriptions of Analysis Scripts
**01_QC_and_Cluster** - General setup and clustering to reproduce the Figures and analysis.

**02_Figure_1** - Script for reproducing Figure 1 and Supplementary Figure 1-2.

**03_Figure_2** - Script for reproducing Figure 2 and Supplementary Figure 3.

**04_Figure_3** - Script for reproducing Figure 3 and Supplementary Figure 4-5.

**CosMx_Nanostring.R** -  Script to reproduce CosMX analysis

**Cibersort_MASLD.R** -  Script to reproduce CibersortX deconvolution algorithm of bulk RNA-seq using sc/snRNA-seq data for Figure 6

**DoMultiBarHeatmap** - Function to make a MultiBarHeatmap
